== Description ==

Media Hub Templates adds the ability to have social media posts easily added to your posts.

== Installation ==

**Installation**

1. Upload the entire `single-post-template` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create Post Templates in your theme's folder, as described in Usage

**Usage**

Install the plugin... you will now have the templates available on your posts.
